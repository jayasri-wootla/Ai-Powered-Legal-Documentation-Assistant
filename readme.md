# AI-Powered Legal Documentation Assistant

## Overview

AI-Powered Legal Documentation Assistant is a web-based application that simplifies legal document analysis and generation using Artificial Intelligence. The system enables users to upload legal contracts, analyze their content, detect important clauses, identify potential risks, receive improvement suggestions, interact with an AI chatbot, generate reports, and create new legal documents automatically.

The application is built using Python, Flask, SQLite, HTML, CSS, and the Groq API.

---

## Features

- User Registration and Login
- Secure Authentication System
- Upload Legal Documents (PDF)
- AI-Based Document Summarization
- Legal Clause Detection
- Risk Analysis
- Document Improvement Suggestions
- AI Chat Assistant
- PDF Report Generation
- Automatic Legal Document Generation
- User-Friendly Dashboard
